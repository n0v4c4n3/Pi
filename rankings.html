Sobre los rankings

Un ranking o ranquin (anglicismo propuesto) es una relación entre un conjunto de elementos tales que, para uno o varios criterios, el primero de ellos presenta un valor superior al segundo, este a su vez mayor que el tercero y así sucesivamente, permitiéndose que dos o más elementos diferentes puedan tener la misma posición. El orden se refleja asignando a cada elemento un ordinal, generalmente números enteros positivos.

Los rankings no representan estrictamente que una pelicula o un usuario sea mejor que el otro.
